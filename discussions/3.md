## Encapsulation

> "Hiding of data."

Process of hiding:

- Properties (variables).
- Behavior (functions).

Done using access modifiers.

- `public`.
- `protected`.
- `private`.
- `friend`.

`friend` violates the concept of object oriented programming.


## Inheritence

Process by which one object gains the properties of another objects.

- Generalization
  - is/are relation.
- Composition
  - has/made-of relation.
- Aggregation
  - has/are relation.
